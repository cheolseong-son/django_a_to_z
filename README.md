이 프로젝트는 무엇을 위한 것인가

2. 프로젝트를 시작하는 방법

프로젝트를 설치, 사용하기 위해 필요한 전제조건이 있는가
어떻게 설치, 사용, 테스트하는가

4. 외부 리소스 정보

프로젝트 내에 포함된 외부의 코드나 리소스의 정보를 포함하는 것이 좋다.

결론
